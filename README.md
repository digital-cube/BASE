# **Base REST API**

* Version 1.0
* [DigitalCube](http://digitalcube.rs/)

### Requirements

* Python >= 3.5
* [PIP](https://bootstrap.pypa.io/get-pip.py)
* mysql

### Installation

* clone the project with git
* enter project directory
* install it with pip:

    `pip3 install .`
    `pip3 install -r requirements.txt`

Easy create REST api for custom application
